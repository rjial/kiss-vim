# kiss-vim
Vim with huge features, ruby, lua, python2, gui (gtk2) support for KISS Linux

**remove vim package first!!**
---
```
git clone https://github.com/tuxsenpaai/kiss-vim
export KISS_PATH=$KISS_PATH:/path/to/kiss-vim

kiss b vim-full
kiss i vim-full
```
